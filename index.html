Noise Variations

The EMA is now accepting digital submissions of Variation Applications via its dedicated email: noise@ema.co.tt. Applicants can download the fillable PDF application form included at the link below under Resources, complete and submit with all required attachments via email. Please note, if an application does not include all the required accompanying documents and/or is not submitted by 4:30 p.m. on the deadline date it will not be accepted.

Application Process
1
Publish Notices
At least 35 days before the event date in any one daily newspaper for 2 consecutive days.
2
Fill out the Application
The Applicant is the owner or Manager of the property.
3
Pay the Fee
Single event — $250
Multiple events — $1,000
4
Submit the Application
Applications must be submitted at least 28 days before the event date.
